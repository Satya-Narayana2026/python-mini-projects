 Python Dice Roller Project
 

A beginner-friendly Python project that simulates rolling dice. This project is built in multiple versions, with each version adding
new features and helping learners understand Python concepts step by step.


###  Version 1: Simple Dice Roller
**File:** `v1_simple_dice_roller.py`

Features:
- Rolls two dice.
- Allows the user to roll repeatedly.
- Handles invalid input.
- Exits when the user chooses to quit.


###  Version 2: Multiple Dice Roller
**File:** `v2_multiple_dice_roller.py`

Features:
- Roll any number of dice.
- Displays the result of each die.
- Counts how many times the user rolled the dice.



###  Version 3: Multiple Dice Roller with Sum
**File:** `v3_multiple_dice_sum.py`



Features:
- Roll multiple dice.
- Displays each dice value.
- Calculates and displays the total sum.
- Tracks the number of game rounds played.
- Validates the number of dice entered.

---

###  Version 4: Dice Roller with History
**File:** `v4_dice_roller_history.py`

Features:
- Roll multiple dice.
- Stores every roll in history.
- Displays all previous rolls when exiting.
- Tracks total rounds played.
- Uses a dictionary to maintain roll history.



## Python Concepts Covered

This project is designed for beginners who want to learn Python through a simple and practical project. Each version introduces new concepts and improvements, making it easy to understand how projects evolve over time.
